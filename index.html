Detailed User Story — AP DMS
User Story
As an Accounts Payable Clerk, I want to submit an invoice digitally with validation, So that I can avoid manual rejection cycles.
Acceptance Criteria
  • System validates PO reference against database at submission.
  • Mandatory fields (Vendor, Amount, Date) must be completed to submit.
  • Automatic escalation if approval is not actioned within 48 hours.
  • Full audit trail logs for all approval/rejection actions.
